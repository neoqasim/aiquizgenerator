

# 🧠 AI Quiz Generator (Flask + Groq API)

A simple AI-powered web app that generates MCQ quizzes from any topic using Groq LLM API.

---

## ⚙️ Setup Instructions

### github link 

```bash
git clone https://github.com/neoqasim/aiquizgenerator 
````


### 1. Clone / Extract Project
```bash
cd ai-quiz-generator
````

---

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Add API Key

Create a `.env` file in the root folder:

```env
GROQ_API_KEY=your_api_key_here
```

---

### 5. Run the Project

```bash
python app.py
```

---

### 6. Open in Browser

```
http://127.0.0.1:5000/
```

---

## 🧠 Done

Enter any topic and the AI will generate a 5-question MCQ quiz.
